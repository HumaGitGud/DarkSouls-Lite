# Dark Slop Souls 

A retro low-poly first-person dark fantasy game built with **Rust + Bevy**. Explore a vast, fog-drowned kingdom, face what stirs in its ruins, and discover where its paths lead.

## Prerequisites

- **Rust** — install from [rust-lang.org/tools/install](https://www.rust-lang.org/tools/install)
- **Windows**: Visual Studio C++ build tools (installed automatically by the Rust installer)

## Build & Run

```bash
cargo run
```

First build compiles Bevy (~5–10 min); later builds are ~15s.

## Controls

| Key / Button | Action |
|---|---|
| `W A S D` | Move |
| `Shift` | Sprint |
| `Left Ctrl` | Dodge roll |
| `Space` | Jump |
| `Mouse` | Look around |
| `Left Click` | Grab cursor / use selected item |
| `Right Click` | Block |
| `Q` | Use equipped artifact |
| `E` | Cycle artifact |
| `Scroll Wheel` | Cycle hotbar item |
| `R` | Interact / rest |
| `Esc` | Pause / release cursor |

## Stack

[Rust](https://www.rust-lang.org/) + [Bevy 0.15](https://bevyengine.org/) — pure code, no editor, low-poly primitives only.
